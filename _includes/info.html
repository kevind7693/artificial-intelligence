Machine Learning

Foundation of AI

All AI requires an application called Machine Learning in order to function like an AI. Machine Learning is what allows systems to learn from data collect, identify and analyze problems, and collected from the data to use to make decisions all on its own with minimal human interactions. Machine Learning contains algorithms and uses 2 types of techniques that helps make an AI an AI. Those 2 techniques are called Supervised Learning and Unsupervised Learning.

Intentionality

One of the 3 qualities

One of the 3 qualities that make an AI. Intentionality is the ability to make decisions using real time data and be able to Quickly makes the most optimal decision based on the data it has collected.This can be harmful to humans if the system is programmed poorly or programmed to harm people in the first place.

Intelligence

One of the 3 qualities

One of the 3 qualities that make an AI. Intelligence is allowing the system to be able to take in data and look for trends/patterns that happens which then it uses to base its decisions off of. The intellect of the AI is based solely on the programmers’ written code and into the AI.

Adaptability

One of the 3 qualities

One of the 3 qualities that make an AI. Adaptability is the ability to learn and adapt as it takes in new information and makes decisions. Being able to constantly take in new information, learn, and adapt can be seem as the most important qualities in an AI since its what makes an AI think like a human.
