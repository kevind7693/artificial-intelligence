Machine Learning

All AI requires an application called Machine Learning in order to function like an AI. Machine Learning is what allows systems to learn from data collect, identify and analyze problems collected from the data to use to make decisions all on its own with minimal human interactions. Machine Learning contains algorithms and uses 2 types of techniques that helps make an AI an AI. Those 2 techniques are called Supervised Learning and Unsupervised Learning.
