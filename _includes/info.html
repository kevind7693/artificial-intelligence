Machine Learning

Foundation of AI

All AI requires an application called Machine Learning in order to function like an AI. Machine Learning is what allows systems to learn from data collect, identify and analyze problems, and collected from the data to use to make decisions all on its own with minimal human interactions. Machine Learning contains algorithms and uses 2 types of techniques that helps make an AI an AI. Those 2 techniques are called Supervised Learning and Unsupervised Learning.

Supervised Learning

Technique 1

This technique builds models that makes predictions based on information gathered in the presence of uncertainty. In addition it takes a known set of input data and known responses to that certain data which then it trains a model to predict a reasonable prediction in response to that data. This is used to predict future outcomes. Supervised Learning uses 2 more technique called classification and regression.

Classification

Classification groups input data into certain categories. Its also used to predict discrete responses. Common algorithms used in order to perform classification includes Support Vector Machine(SVM), Boosted and bagged decision trees, k-nearest neighbor, Naive Bayes, discriminant analysis, logistics regression, and neural networks.

Regression

Regression predicts continuous responses and is used when you are working with a data range or when your response is a real number. Common algorithms used in order to perform regression includes Linear model, Nonlinear model, regularization, stepwise regression, Boosted and bagged decision trees, neural networks, and adaptive neuro-fuzzy learning.

Unsupervised Learning

Technique 2

Unsupervised Learning is used for finding hidden patterns or intrinsic structures in data, then drawing inferences from a dataset consisting of input data without label responses. Clustering being the most common type of unsupervised learning technique.

Clustering

Unsupervised Learning

Clustering is a type of unsupervised learning that uses exploratory data analysis to find hidden patterns and groupings in data. Common algorithms used in Clustering includes k-means and k-medoids, hierarchical clustering, Gaussian mixture model, hidden Markov model , Self-organizing maps, fuzzy c-means clustering, and subtractive clustering.
